This is the Star War search engine. Learn information on different characters in the Star Wars universe. This application utilizes the free SWAPI* API to get and display information on the user's query.

Functionality

still developing
currently the People and Films searches work. it is also possible to search without inputting a full name. For example searching for "luke" under the "People" tab will return "Luke Skywalker". Under the "Films" tab, searching for "empire" will return "Empire Strikes Back".

*API only has data for episodes 1-6.
